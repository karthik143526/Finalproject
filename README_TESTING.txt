════════════════════════════════════════════════════════════════════════════════
                          TROUBLESHOOTING QUICK HELP
════════════════════════════════════════════════════════════════════════════════

Common Issues & Fixes:

Server not running:
  → Start XAMPP Apache & MySQL services
  → Verify http://localhost/finalproject opens in browser

Chrome error:
  → pip install webdriver-manager --upgrade
  → Install Chrome browser

Dependencies error:
  → pip install -r requirements.txt --upgrade
  → Restart terminal/command prompt

Database error:
  → Import eco2.sql in phpMyAdmin
  → Verify 'eco' database exists
  → Check MySQL is running

Test timeout:
  → Edit comprehensive_e2e_tests.py
  → Increase DEFAULT_TIMEOUT and LONG_TIMEOUT values
  → Re-run tests

For more help: See TEST_GUIDE.md
